This site is a personal project created while studing Angular and Java.

I started learning JAVA 17, then Spring Boot and created some Rest API, but I could only see the results using POSTMAN.

I thought, what about learning some front-end framework? So I could see better the result. Then I started learning ANGULAR.

The first part of the project was: Create a REST API with a CRUD with one table called Beers

The Fron-End is this about page and a List of Beers where you can access the detail.

After I finished Angular Forms, I created the "Add Beer" and added the delete and Update functions.
